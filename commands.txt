npm init       - This command will help us to setup package.json
npm init -y    - This command will help us to setup package.json without any prompts
npm install express - This command will help us to get express module from npmjs.com
npm install - This will help you to install all the packages present in package.json file
node app.js - To run the node file / application.
node app - We dont have to provide extension, node would assume that we are trying to run .js file.

In package.json we can include commands in scripts.
If its node specific, we can run using npm nodeSpecificCommand
If its not node specific, we can run using npm run notNodeSpecificCommand
